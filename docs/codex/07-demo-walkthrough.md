# 最终演示脚本与验收清单

## 1. 使用说明

- 默认演示环境：真实前端 + 真实后端 + 真实 MySQL
- mock 环境仅用于回退冒烟，不作为正式答辩路径
- 三类演示账号统一密码：`demo`
- AI 助手当前是占位能力，统一返回 `50100 / AI assistant is disabled in V1`

### mock 环境最小启动说明

1. 进入 `frontend`
2. 执行 `npm run dev -- --mode mock`
3. 确认登录页出现 Mock 环境说明
4. 确认顶部出现 `Mock 演示环境`
5. 确认右上角出现“重置本地 Mock 数据”按钮
6. 确认日志页与 AI 助手页可打开且不报错

## 2. risk-demo

### 角色

- `risk-demo`
- 风控人员

### 步骤

1. 登录后进入“风险数据”，新增一条业务并填写全部启用指标值
2. 编辑刚创建的业务，确认保存后状态与提示正常
3. 从列表点击“去评估”执行评估
4. 打开评估详情，查看总分、风险等级、`indexResults`
5. 进入“预警管理”，找到对应预警并提交处理意见、处理结果、下一状态
6. 回到“仪表盘”与“统计分析”，刷新后查看预警与统计是否回流
7. 切换到 `admin-demo` 修改一条相关规则，再回到旧评估详情确认历史分项结果不漂移

### 预期页面结果

- 风险数据新增/编辑成功，错误提示优先展示后端中文业务错误
- 评估详情能看到历史固化的 `indexResults`
- `MEDIUM/HIGH` 评估会自动生成预警，`LOW` 不生成
- 预警处理后，仪表盘与统计页数据同步变化
- 旧评估详情不受后续规则修改影响

### 预期日志结果

- 日志页可筛到“风险数据 / 新增、编辑、删除（如有）”
- 日志页可筛到“风险评估 / 执行”
- 日志页可筛到“预警 / 处理”

### 失败时先检查什么

1. `backend` 是否已连接真实 MySQL 并成功启动
2. 风险数据是否补齐全部启用指标值
3. 当前指标规则是否覆盖该业务的全部指标值区间
4. 日志页是否以 `admin-demo` 登录后再按模块/操作类型筛选

## 3. manager-demo

### 角色

- `manager-demo`
- 管理人员

### 步骤

1. 登录后查看仪表盘，确认角色文案与图表显示正常
2. 进入“预警管理”，打开一条预警详情
3. 查看该预警的处理记录时间线
4. 进入“统计分析”，切换筛选条件观察图表变化
5. 验证自己没有处理预警、用户管理、指标规则写操作权限

### 预期页面结果

- 可正常查看仪表盘、预警、统计页
- 无处理权限时页面不应出现空白页或未捕获异常
- 若通过接口或越权路径尝试处理预警，应返回明确无权限提示

### 预期日志结果

- 管理人员的只读行为不会新增业务写日志
- 若尝试越权处理，后端应返回 `40300`，前端保持自然错误提示

### 失败时先检查什么

1. 当前登录账号是否真的是 `manager-demo`
2. 路由守卫与菜单权限是否正确读取 `roleCode`
3. 后端接口是否返回 `40300` 而不是 `50000`

## 4. admin-demo

### 角色

- `admin-demo`
- 系统管理员

### 步骤

1. 登录后进入“指标规则”，完成一次指标新增、编辑、启停
2. 在同页完成一次规则新增、编辑、删除
3. 进入“用户管理”，完成用户新增、编辑、启停、删除
4. 验证不能停用或删除当前登录用户
5. 进入“操作日志”，按模块与操作类型筛选真实写操作
6. 进入“AI 助手”页发送一次占位请求

### 预期页面结果

- 指标与规则写操作都成功落到真实后端
- 用户管理允许改姓名/手机号/角色，不强制每次编辑都改密码
- 用户角色选择语义明确为单角色覆盖
- 当前登录用户保护有效
- AI 助手页返回禁用提示，但页面行为稳定

### 预期日志结果

- 可筛到“指标规则 / 新增、编辑、启用、停用”
- 可筛到“用户 / 新增、编辑、启用、停用、删除”
- 可筛到“系统 / 登录、登录失败、退出”

### 失败时先检查什么

1. 日志页筛选条件是否使用 `moduleName + operationType + operator`
2. 用户编辑表单是否误把密码设为必填
3. 新增指标若无法回收，是否已保持停用避免影响业务权重

## 5. 最终验收清单

- `risk-demo / manager-demo / admin-demo` 三类账号都能正常登录
- 登录失败、登录成功、退出三类日志都能在日志页查到
- 风险数据 -> 评估 -> 预警 -> 统计回流链路完整可演示
- 管理人员只读路径稳定，越权操作返回明确错误
- 管理员配置、用户管理、日志筛选、AI 占位页全部可演示
- 默认前端环境不再出现“部分模块过渡中”类文案
- mock 环境仍可独立启动，但不影响默认真实链路
